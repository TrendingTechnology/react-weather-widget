## How to Use

1. Grab the repo

2. Under the directory, use npm for yarn

  `npm install`
  
  or
  
  `yarn install`
  
3. Use the start command to start up a server the widget should pop in a new tab of your browser.

  `npm start`
  
  or
  
  `yarn start`
